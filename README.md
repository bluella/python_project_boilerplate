# Claude Rules for Python Boilerplate

Opinionated `.claude/rules/` for Python projects.

Some ideas stolen from [@boris_again](https://t.me/boris_again) and [@danokhlopkov](https://t.me/danokhlopkov).

## Useful tools to pair with Claude

- [cmux](https://github.com/sidhant-sriv/cmux) — conveniently run multiple Claude Code agents in parallel
- [plannotator](https://github.com/nicobailon/plannotator) — annotate and track Claude's plans
- [spokenly](https://spokenly.app) — best voice input, which works even locally
